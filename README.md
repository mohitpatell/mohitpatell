## 🧠 Current Focus

### 🔥 [klyne](https://github.com/klyne-ai/klyne)

A **local-first MCP server** for Claude Code and Codex CLI. Reads the JSONL transcripts your AI already writes locally — and gives you back the context that `/compact`, rate limits, and fresh sessions destroy.

- Recovers original pre-`/compact` turns
- Generates deterministic handoff prompts between sessions
- Surfaces context health, token timeline, and cache-reuse stats
- Searches across every past Claude / Codex session on your machine
- **No cloud. No proxy. No telemetry. Read-only by design.**

🎬 [Watch the 5-minute demo](https://youtu.be/2NglEGlq3Ns)
